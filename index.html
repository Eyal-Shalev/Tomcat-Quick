Tomcat Quick

Adds a menu button to the top right mac menu with an option to start & stop tomcat.

View the Project on GitHub Eyal-Shalev/Tomcat-Quick

About

This app will make you life easier when developing with Tomcat (guaranteed).

Installation

  1. Download the Tomcat Quick.dmg from the downloads page.
  2. Open the Disk image.
  3. Copy the Tomcat Quick.app to your Applications folder.
  4. Launch the app.
